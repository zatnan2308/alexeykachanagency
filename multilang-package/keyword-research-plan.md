# Keyword Research Plan — Alexey Kachan Agency

## 8 языков × 6 услуг + общие страницы

> **Назначение:** этот документ — рабочая таблица ключевых запросов для каждой страницы каждого языка. Используйте при написании meta title, meta description, H1, контента страниц и при настройке Google Ads. Регулярно обновляйте на основе данных из Google Search Console и Ahrefs/SEMrush.

> **Метод:** запросы подобраны на основе типичных паттернов поиска в каждом регионе с учётом интента (информационный / коммерческий / транзакционный) и частотности. Цифры приблизительные — обязательно сверяйте с актуальным Keyword Planner для каждого региона перед запуском кампаний.

> **Условные обозначения:**
> - **MV** — Monthly Volume (примерный месячный объём поиска для региона)
> - **KD** — Keyword Difficulty (сложность ранжирования, 0-100, ниже = легче)
> - **Intent**: I = informational, C = commercial, T = transactional
> - **Priority**: 🟢 Primary, 🟡 Secondary, ⚪ Long-tail

## 1. ОБЩАЯ СТРАТЕГИЯ ПО ЯЗЫКАМ

| Язык | Конкуренция | Платёжеспособность | Барьер входа | Стратегия |
|------|------------|---------------------|--------------|-----------|

| **DE** | 🔴 Высокая | 🟢 Очень высокая | 🟡 Средний (язык) | Премиум-сегмент, чёткий процесс, GDPR-fokus |

| **ES** | 🟡 Средняя | 🟡 Средняя | 🟢 Низкий | Объём + LATAM рынок, ценовая доступность |
| **IT** | 🟡 Средняя | 🟢 Высокая | 🟡 Средний | Сегмент SMB, Made in Europe качество |
| **RO** | 🟢 Низкая | 🟡 Средняя | 🟢 Низкий | **Лучший ROI** — низкая конкуренция, растущий рынок |
| **UK** | 🟡 Средняя | 🟡 Средняя | 🟢 Очень низкий | Домашний рынок, прямые лиды, патриотический фактор |
| **RU** | 🔴 Высокая | 🟡 Средняя | 🟢 Низкий | Международная RU-аудитория (без географии РФ) |

### Рекомендованный порядок наполнения по приоритету ROI

1. **EN** — основной trafic-driver, самый большой пул (но самая высокая конкуренция)
2. **RO** — лучший quick-win, низкая конкуренция, растущий рынок
3. **UK** — домашний рынок, лояльная аудитория, легко ранжироваться
4. **DE** — премиум-сегмент, высокий чек, окупает затраты на качественный перевод
5. **IT, FR** — стабильные европейские рынки
6. **ES** — большой пул, но нужно решить: фокус на Spain или LATAM
7. **RU** — спорный (этический и санкционный) рынок; решение — за владельцем

## 10. ИНСТРУМЕНТЫ ДЛЯ ВЕРИФИКАЦИИ

Перед запуском обязательно сверьте все цифры через:

1. **Google Keyword Planner** — бесплатно, базовая истина для каждого региона
2. **Google Trends** — относительные тренды и сезонность
3. **Ahrefs / Semrush** — точные KD и SERP-ситуация (платно, но для агентства must-have)
4. **Mangools KWFinder** — дешёвая альтернатива, неплохо работает для малых рынков (RO, UK)
5. **Moz Keyword Explorer** — бесплатные 10 запросов/день, хорошо для проверки
6. **Search Console** (после запуска) — реальные запросы, по которым показывается сайт

### Цикл оптимизации

1. **Месяц 1**: Запустить с этими ключами, отслеживать импрешны/клики в GSC
2. **Месяц 2-3**: Найти "low hanging fruit" — запросы, по которым есть импрешны на позициях 8-20, оптимизировать контент под них
3. **Месяц 4+**: Дополнить блог по запросам, по которым есть спрос, но недостаточно покрытия
4. **Каждый квартал**: Полная ревизия ключевых страниц на основе данных

## 11. БЫСТРАЯ ПАМЯТКА ПО МЕТА-ТЕГАМ

**Title формула (рабочая):**  
`[Primary Keyword] — [Benefit / Secondary KW] | Brand Name`

**Description формула:**  
`[Action verb] [primary KW] [for whom]. [Differentiator / numbers]. [Trust signal]. [CTA].`

**Длины:**
- Title: 50-60 символов (до 580px)
- Meta description: 140-160 символов
- H1: 20-70 символов, 1 на страницу, содержит primary keyword
- Slug: 3-5 слов через дефис, lowercase, без stop-words

**Чек-лист перед публикацией страницы:**
- [ ] Primary keyword в title (ближе к началу)
- [ ] Primary keyword в meta description
- [ ] Primary keyword в H1
- [ ] Primary keyword в первых 100 словах контента
- [ ] Primary keyword в URL (slug)
- [ ] Secondary keywords в H2/H3
- [ ] Synonyms и LSI keywords рассыпаны по тексту (естественно)
- [ ] Внутренние ссылки на 3-5 связанных страниц
- [ ] FAQ-блок отвечает на 3-5 People Also Ask вопросов
- [ ] Schema.org разметка соответствует типу страницы
- [ ] Hreflang на все 8 языковых версий + x-default
- [ ] OG-image уникальный для страницы и языка
