## Обзор проекта

Happy Sofa YEG — семейный сервис химчистки мягкой мебели в Эдмонтоне (Альберта) с уверенным брендовым посылом «Life is Messy. We Make it Pure.». Глубокая паровая чистка диванов, угловых модулей, матрасов и ковров экологичными средствами, безопасными для детей и животных.

## Что мы сделали

- Смелый hero — «Life is Messy. We Make it Pure.» — с двумя CTA (бесплатная заявка + WhatsApp)
- Блок доверия: 500+ домов, рейтинг 5★, 100% эко, запись день-в-день
- Карточки услуг: диваны и угловые, санитизация матрасов, уборка после животных, обновление ковров
- Запись в 4 шага («How It Works»)
- Галерея до/после с короткими видео
- FAQ + форма заявки, интеграция с WhatsApp для мгновенного контакта

Сделано на WordPress с ACF Pro — каждый блок страницы редактируется владельцем.
