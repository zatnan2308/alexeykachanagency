## Project overview

Wright Group Event Services is a family-owned event rental company supplying tents, tables, lighting and full-service production for weddings, corporate events and private parties. The brief was a presence site that captured inquiries reliably without a heavy CMS overhead.

## What we built

- Multi-step "Make event request" form routed by event type
- Service catalogue with photo galleries and on-page testimonials
- Live counters (happy customers, events organised, working days)
- Google Reviews integration with star aggregator
- Blog section for SEO content and event tips
- Mobile-first responsive layout for on-the-go event planners
