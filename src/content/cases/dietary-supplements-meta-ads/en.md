## Project overview

The client had been through three media buyers in a year. Each had promised quick wins; each had left the account in worse shape. The brief: don't promise anything until the audit is complete, then run a controlled test before scaling.

## What we did

- 12-page audit of the existing account — pixel events, audience hygiene, creative archive, bid strategy
- Identified three structural issues (broken Pixel events, CBO budget mismatch, audience cannibalisation)
- Fixed Pixel + CAPI implementation to restore tracking accuracy
- Launched a single controlled test ad set with $4/day budget to validate the rebuild
- Scaled only after the test ad set hit ROAS 5×+ for 5 consecutive days

## The result

The validation test delivered **72 purchases at $1.80 cost-per-purchase, ROAS 5.10×** on $129.56 spend. The client signed an ongoing scaling contract on the strength of the test alone.
