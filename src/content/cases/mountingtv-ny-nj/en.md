## Project overview

Mountingtvservice handles TV mounting, fireplace TV installation, soundbars, wire concealment and Samsung Frame mounting across New York and New Jersey. The hero question every visitor asks first is "what does this cost?" — so the site answers it before scrolling.

## What we built

- Hero with "What size is the TV?" inline pricing tiles ($49 up to $139)
- Service tiles for each install type with example imagery
- "Our works" gallery grid with 18+ completed installations
- Wall-mount bracket recommendations (tilt, full-motion, etc.)
- "Why people choose us" comparison block
- TaskRabbit-integrated reviews badge with star count
- "Drop a line" inquiry form for non-standard jobs
